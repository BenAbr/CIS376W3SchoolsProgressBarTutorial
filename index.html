Welcome to the Progress Bar Tutorial

By Ben Abroms

In this tutorial, we will learn how to make a bar that shows visual progress made when pressing a button.

10%

The Progress Bar above is the one we will be creating for this tutorial.

First, we will need to make two div objects. One will be labeled with a myProgress ID and the other will be myBar.

<div id="completedProgressbar">
<div id="myBar">10%</div>
</div>

The 10% that was put of on myBar gives us a visual indicator of where we are starting. Also, will need to add a button.

<button onclick="move()">Run</button>

When we press the button, it will activate a JavaScript, which we will write later, to run our progress bar.

Next, we'll need to make a style in CSS for the design of the progress bar itself and the under part.

.myProgress {
width: 100%;
background-color: #ddd;
}
#myBar {
width: 10%;
height: 30px;
background-color: #4CAF50;
text-align: center;
line-height: 30px;
color: white;
}
10%

We have our bar's starting place and the button, but neither of them do anything until we attach a JavaScript to them.

Now we need to make the Javascript code for the bar to work.

function move() {
var elem = document.getElementById("myBar");
var width = 10;
var id = setInterval(frame, 10);
function frame() {
if (width >= 100) {
clearInterval(id);
} else {
width++;
elem.style.width = width + '%';
elem.innerHTML = width * 1 + '%';
}
}
}

The Javascript find the right element by searching for the id "myBar" then is creates two variables, width and id. The id variable is defined by a function setInterval that takes in frame(return of a different function) and 10. In the frame function, it increases myBar's width element by 1% every so units of time. This makes the progress bar move to its end and stop.

Here is the final result! Your brand new Progress bar!
